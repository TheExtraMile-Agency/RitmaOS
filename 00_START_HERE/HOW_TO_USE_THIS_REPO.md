# How To Use This Repo

Use this repository as a layered research and product-development system.

1. Store raw source material in `02_RAW_MODULES/`.
2. Extract reusable frameworks into `03_EXTRACTED_FRAMEWORKS/`.
3. Draft product implications in `04_PRODUCT_SPEC_DRAFTS/`.
4. Record durable decisions in `06_DECISIONS_LOG/DECISIONS.md`.
5. Track unresolved synthesis work in `07_SYNTHESIS_QUEUE/`.
6. Update indexes and `PACKAGE_MANIFEST.md` when files are added or moved.

Raw module text is immutable. Analysis belongs outside the raw source layer.
