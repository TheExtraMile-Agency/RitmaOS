# Principles

- Folder = memory.
- Prompt = direction.
- Spec = leverage.
- Workflow = staged folders.
- Output = reusable context.
- Review = reliability.
- Memory = decisions and lessons learned.
- AI = component, not the whole system.
- Architecture is the product.
- Filesystem structure can act as plain-English orchestration.

These are working principles for research and synthesis. They are not final product doctrine.
