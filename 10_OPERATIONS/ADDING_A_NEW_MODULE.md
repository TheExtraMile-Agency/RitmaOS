# Adding A New Module

1. Create an inbox file in `05_NEW_MODULES_INBOX/`.
2. Paste raw module text exactly.
3. Do not summarize or clean up the raw text.
4. Commit the inbox addition separately.
5. After review, move the accepted raw module into `02_RAW_MODULES/`.
6. Update `02_RAW_MODULES/00_RAW_MODULES_INDEX.md`.
7. Regenerate `PACKAGE_MANIFEST.md`.
