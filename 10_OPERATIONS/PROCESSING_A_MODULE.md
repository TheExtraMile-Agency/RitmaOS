# Processing A Module

1. Confirm the raw module exists in `02_RAW_MODULES/`.
2. Leave raw source text untouched.
3. Extract reusable frameworks into `03_EXTRACTED_FRAMEWORKS/`.
4. Update `03_EXTRACTED_FRAMEWORKS/00_FRAMEWORKS_INDEX.md` when adding framework files.
5. Track pending questions in `07_SYNTHESIS_QUEUE/`.
6. Commit extraction work separately from raw source additions.
